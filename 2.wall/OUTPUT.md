# 2.wall
플레이어 시점으로부터 수평시야각(FOV_H)범위에서 빛줄기를 하나씩 쏴서 벽의 위치와 거리를 구한다.

빛줄기는 스크린 가로 해상도(SX) 픽셀 수만큼 쏜다. 본 챕터에서 플레이어의 시점은 점(dot) 기준이다.

### 프로그램 컴파일
```
shell $ cc -Wall *.h *.c -lm
```

### 프로그램 입력 예시
```
shell $ ./a.out 1.5 3.5 355
```
